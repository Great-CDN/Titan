
# 培养计划：系统方向



## 课题一：Crash

捕捉Crash信号，获取Crash堆栈，自行处理



## 课题二：性能调优

工具：Valgrind，perf，SystemTap，BPF

系统：系统调用，信号，Hook

CPU：亲和性，调度，上下文切换，NUMA，中断，缓存

内存：大页，页面缓存，swap

IO：IO调度，DMA

磁盘：文件系统，块，挂载

网络：分包，缓冲区配置，RPS，Offload



## 课题三：进程

结构：ELF文件，PE文件，内存布局

调试：审计，跟踪，修改

控制：fork、wait、exec

通信：管道、消息队列、共享内存，信号量

竞争：锁、原子操作
